### Phase 36: SVG Export with Named Layers (6 requirements)

- [x] **SVG-01**: Export as SVG option for any element (property panel button)
- [x] **SVG-02**: SVG layers have meaningful names based on element structure
- [x] **SVG-03**: Consistent naming convention documented for each element type
- [x] **SVG-04**: When importing SVG for element, auto-match layers by name
- [x] **SVG-05**: Validation on import (error if required layers missing, warning if extra)
- [x] **SVG-06**: Layer naming ensures re-imported SVGs work immediately

### Phase 37: Font Management System (7 requirements)

- [x] **FONT-01**: User-selected fonts folder via file picker
- [x] **FONT-02**: Manual rescan on user request
- [x] **FONT-03**: All discovered fonts appear in font selection dropdowns
- [x] **FONT-04**: Settings UI with folder selection and rescan trigger
- [x] **FONT-05**: Display selected folder path with copy button
- [x] **FONT-06**: Export only bundles fonts actually used in the project
- [x] **FONT-07**: Custom font dropdown with preview (fonts shown in their typeface)
