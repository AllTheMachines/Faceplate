# Requirements: VST3 WebView UI Designer

**Defined:** 2025-01-23
**Core Value:** Visually design a plugin UI and export code that works in JUCE WebView2 without manual fixups.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Canvas

- [x] **CANV-01**: Canvas with configurable dimensions (e.g., 800x600) — Phase 1
- [x] **CANV-02**: Pan with spacebar+drag — Phase 1
- [x] **CANV-03**: Zoom with scroll/pinch — Phase 1
- [x] **CANV-04**: Click to select single element — Phase 3
- [x] **CANV-05**: Shift+click for multi-select — Phase 3
- [x] **CANV-06**: Marquee (drag) selection — Phase 3
- [x] **CANV-07**: Delete selected elements — Phase 3
- [x] **CANV-08**: Background color/gradient or image — Phase 1
- [x] **CANV-09**: Foreground/overlay images — Phase 4
- [x] **CANV-10**: Element z-order (layering) — Phase 4

### Palette

- [x] **PALT-01**: Component palette with categorized controls — Phase 4
- [x] **PALT-02**: Drag from palette onto canvas — Phase 4
- [x] **PALT-03**: Built-in SVG library (knobs, sliders, buttons, meters, labels) — Phase 4
- [x] **PALT-04**: Custom SVG import with layer name detection (indicator, thumb, track, fill) — Phase 4

### Manipulation

- [x] **MANP-01**: Move elements by dragging — Phase 5
- [x] **MANP-02**: Resize elements with handles — Phase 5
- [x] **MANP-03**: Arrow key nudge (1px, shift+10px) — Phase 5
- [x] **MANP-04**: Snap to grid — Phase 5
- [x] **MANP-05**: Copy/paste elements (Ctrl+C/V) — Phase 6

### Properties

- [x] **PROP-01**: Property panel for selected element — Phase 5
- [x] **PROP-02**: Direct numeric input for all values — Phase 5
- [x] **PROP-03**: Color pickers for color properties — Phase 5
- [x] **PROP-04**: Element name field (becomes ID in export) — Phase 5
- [x] **PROP-05**: Parameter ID field for JUCE binding — Phase 5

### History

- [x] **HIST-01**: Undo (Ctrl+Z) — Phase 3
- [x] **HIST-02**: Redo (Ctrl+Y) — Phase 3

### Persistence

- [x] **PERS-01**: Save project as JSON file — Phase 7
- [x] **PERS-02**: Load project from JSON file — Phase 7

### Export

- [x] **EXPO-01**: Export JUCE WebView2 bundle (index.html, styles.css, components.js, bindings.js, bindings.cpp) — Phase 8
- [x] **EXPO-02**: Export HTML preview with mock values — Phase 8
- [x] **EXPO-03**: Generated IDs use element names — Phase 8

### UI/UX

- [x] **UIUX-01**: Three-panel layout (palette, canvas, properties) — Phase 1
- [x] **UIUX-02**: Dark theme — Phase 1
- [x] **UIUX-03**: Delete key shortcut — Phase 6

### Tech Stack

- [x] **TECH-01**: React 18 + TypeScript — Phase 1
- [x] **TECH-02**: Vite for build — Phase 1
- [x] **TECH-03**: Zustand for state management — Phase 1
- [x] **TECH-04**: @dnd-kit/core for drag-drop — Phase 1
- [x] **TECH-05**: Tailwind CSS for styling — Phase 1
- [x] **TECH-06**: Browser-based (no Electron) — Phase 1

### v1 Element Types

- [x] **ELEM-01**: Knob (arc style, configurable angles/colors/indicator) — Phase 2
- [x] **ELEM-02**: Slider (vertical/horizontal, track/thumb styling) — Phase 2
- [x] **ELEM-03**: Button (momentary/toggle) — Phase 2
- [x] **ELEM-04**: Label (text display) — Phase 2
- [x] **ELEM-05**: Level meter (peak, vertical/horizontal) — Phase 2
- [x] **ELEM-06**: Image (for backgrounds, logos, decorative) — Phase 2

### Reference

- [x] **REF-01**: See docs/SPECIFICATION.md for complete property definitions per element type — Phase 8
