<decisions_made>
**v2.0 Decisions:**
- Make repository private for Pro feature protection
- Use Polar.sh for license validation (no backend needed)
- 7-day cache for license validation with offline tolerance
- Accept client-side gating risk for now; revisit if revenue justifies stronger protection
- VITE_DEV_PRO=true enables Pro features in development
- Pro badges use violet-500 (#8B5CF6) for visual consistency
- isPro field is optional boolean (undefined/false = Free, true = Pro)
- License state excluded from undo/redo history (user-level, not document)
- Hide Pro elements toggle defaults ON for new users
- 50 Pro elements total across 6 categories:
  - ASCII (3): asciislider, asciibutton, asciiart
