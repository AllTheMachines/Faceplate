- Rocker Switch position state layers use opacity toggle

<context>
**Current milestone:** v0.10.0 SVG Styling for Visual Controls
**Goal:** Enable custom SVG artwork for all visual controls (knobs, sliders, buttons, meters)
**Approach:** Layer-based SVG styling with convention-based layer naming

**Layer naming conventions:**
- Knobs: knob-body, knob-pointer, knob-indicator, etc.
- Sliders: slider-track, slider-fill, slider-thumb
- Buttons: button-normal, button-pressed, button-icon
- Switches: switch-body, switch-indicator, switch-on, switch-off

**Tech stack:** React + TypeScript + Vite + Zustand + TailwindCSS + dnd-kit
</context>
