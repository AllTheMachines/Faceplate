## E2E User Flows

### Flow 1: Import SVG → Create Style → Apply → Export ✓

1. User selects slider element
2. PropertyPanel shows ElementStyleSection
3. User clicks "Manage styles..."
4. ManageElementStylesDialog opens with category filter
5. User clicks "Import New Style"
6. ElementLayerMappingDialog opens
7. User drops SVG → auto-detection works
8. User maps layers → clicks Save
9. addElementStyle called → style persisted
10. Style appears in dropdown
11. User selects style → element renders with SVG
12. Export generates styled HTML/CSS/JS

### Flow 2: Save → Load → Styles Persist ✓

1. User creates element styles
2. User clicks Save → elementStyles serialized
3. User closes/reopens app
4. User clicks Load → setElementStyles restores styles
5. Styles appear in PropertyPanel dropdowns
6. Styled elements render correctly

### Flow 3: Load v2 Project → Migration ✓

1. User loads old v2 project with knobStyles
2. migrateV2ToV3 converts to elementStyles
3. setElementStyles loads migrated styles
4. Rotary styles available for knob elements
